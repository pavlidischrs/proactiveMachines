
 Ubuntu 14 instructions 

-- After installing prerequisities from http://caffe.berkeleyvision.org/installation.html
   run the compile commnands:

   $ make {all,test,runtest,pycaffe}   /*  Modify Makefile.config if need it  */

   

-- Then, in examples/st_mnist directory lives our example. Run train_lenet.sh (modify directories if need it) 
   to start training.


lenet_solver.prototxt 		file has the training paramateres 
lenet_train_test.prototxt 	file has the network architecture

class.py			file is for single pictures classification (not commented)
check.py			file can be used to check changes on during trainig or every iteration (not commented)

pics 				folder has some distorted pictures from the train set
